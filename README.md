# Terminal Pomodoro

## Purpose
Create a terminal based rust application to start, save, and notify standard pomodoro timers.  Designed based on the [spotify-tui](https://github.com/Rigellute/spotify-tui#cargo) project.

## Technologies
1. [Rust Lang](https://www.rust-lang.org/)
2. [tui-rs](https://github.com/fdehau/tui-rs)
3. [Pomodoro Technique](https://www.techtarget.com/whatis/definition/pomodoro-technique#:~:text=The%20Pomodoro%20Technique%20is%20a,after%20four%20consecutive%20work%20intervals.)
